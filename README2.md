# 🦠 COVID-19 Vaccination Data Analysis

A **Data Analytics Mini Project** developed using **Python**, **Pandas**, and **Matplotlib** to analyze COVID-19 vaccination data. The project explores vaccination trends, performs data preprocessing, and generates meaningful insights through data visualization.

---

## 📌 Project Overview

This project focuses on analyzing COVID-19 vaccination data to understand vaccination progress across different states. The dataset is cleaned, processed, and analyzed using Python libraries to extract useful insights and visualize vaccination statistics.

The project demonstrates essential **Data Analytics** techniques including data cleaning, exploratory data analysis (EDA), aggregation, and visualization.

---

## ✨ Features

- 📊 Analyze COVID-19 vaccination data
- 🧹 Data Cleaning and Preprocessing
- 📈 Data Visualization using charts
- 🔍 State-wise vaccination analysis
- 📋 Summary statistics of the dataset
- 📉 Identify vaccination trends and patterns

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/your-username/COVID-19-Vaccination-Data-Analysis.git
```

### Navigate to the project folder

```bash
cd COVID-19-Vaccination-Data-Analysis
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```
